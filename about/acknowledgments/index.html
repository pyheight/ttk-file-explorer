    开源库致谢⚓︎

    在这里仅列出主要的几项：

    • tkinter
      Python 的标准 GUI 工具包，为项目提供基础 GUI 支持。

    • ttkbootstrap
      tkinter 和 ttk 的现代风格扩展，带来美观的界面设计。

    • winshell
      简化 Windows 平台桌面管理任务。

    • pystray
      实现系统托盘功能。

    • pyinstaller
      将 Python 项目打包成独立可执行文件。

    提示

    推荐项目⚓︎

    以下是启发我们的优秀项目，推荐大家尝试：

    • Tk-Path-Finder
      带选项卡的 Windows 文件资源管理器，使用 ttkbootstrap 开发。

    • biscuit
      快速、可扩展的本机 IDE（大小 < 20MB）。

    • mkdocs-material
      构建本网站的优雅文档框架。
