    Issue Reporting⚓︎

    Only Latest Versions Receive Active Updates⚓︎

    Encountered an issue or have a suggestion?

    Your feedback is crucial for improving ttk file explorer!

    GitHub Issues Start Discussion Email

    Reporting Requirements⚓︎

    Please provide the following information

    1. Issue Description
      Problem encountered · Frequency and conditions of occurrence

    2. Reproduction Steps
      Step 1 · Step 2 · Observed result

    3. Environment Information
      OS version · Software version

    4. Additional Information
      Screenshots or screen recordings · Error logs (if available) · Contact details (optional)

    Tip

    You can also report via email.

    Reporting Guidelines⚓︎

    Report Type Priority Recommended Channel
    Critical Bug
    Crash/data loss    		 GitHub Issues
    Feature Request
    New feature/improvement suggestion    		 Start Discussion
    Usage Question
    Configuration/operation guidance    		 Start Discussion

    Tip

    We'll address reports as soon as possible. Thank you for your patience and support!

    Pre-submission Checklist⚓︎

    • Searched existing issues
    • Included necessary environment information
    • Provided clear reproduction steps
